Special sprint-related commands which are available only to an Admin To use this command you must have the @SprintAdmin role, be an administrator on the server, or be the server owner. or MC To use this command you must have the @SprintMC role, @SprintAdmin role, be an administrator on the server, or be the server owner. .

Commands #

sprint lock #

MC To use this command you must have the @SprintMC role, @SprintAdmin role, be an administrator on the server, or be the server owner.
/sprint options: lock

Begins a sprint which cannot be cancelled except by a Sprint MC.

Example:

/sprint options: for 30 in 5 endtime 10 lock
@Sprinto#2517 sprint for 30 in 5 endtime 10 lock
Begins a 30 minute locked sprint with 5 minutes to join and 10 minutes to give a word count. Note: “lock” can also appear elsewhere, such as at the start.

See [Sprint] for the other parameters you can add to a /sprint

sprint please #

SprintMCs who add “please” to a sprint command can set it to run for longer, or have slightly increased “ready” time before it starts than would normally be allowed.

Note: This isn’t to force anyone to be polite. It’s an added check so you don’t accidentally run long sprints.

Example:

/sprint options: for 90 minutes please
@Sprinto#2517 sprint for 90 pls
Synonyms
/sprint options: 90 pls
@Sprinto#2517 sprint for 1.5 hours s’il vous plaît

cancel please #

MC To use this command you must have the @SprintMC role, @SprintAdmin role, be an administrator on the server, or be the server owner.

/admin-force-cancel
@Sprinto#2517 cancel pls
Synonyms
/cancel please: True
/sprint options: cancel pls
@Sprinto#2517 cancel ffs
@Sprinto#2517 cancel pleeeeeeeease

Forces a running sprint to end, regardless of how many sprinters have joined.

nudge please #

MC To use this command you must have the @SprintMC role, @SprintAdmin role, be an administrator on the server, or be the server owner.

/admin-force-nudge
@Sprinto#2517 nudge pls
Synonyms
/sprint options: nudge pls

Moves a running sprint on to the next stage immediately, typically only used for testing or debugging purposes.

If a bug occurs and the sprint becomes stuck, @Sprinto#2517 nudge (without the “please”) can also be used by anyone to nudge the sprint along. That’s if Sprinto can successfully detect that the sprint is stuck.
